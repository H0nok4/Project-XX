## 5. 任务系统详细设计

### 5.1 任务分类

**主线任务**
- 推进游戏核心剧情
- 解锁新地图和功能
- 必须按顺序完成
- 奖励丰厚且独特

**支线任务**
- 商人委托任务
- NPC请求任务
- 可选完成
- 提升信誉和关系

**日常任务**
- 每日刷新
- 可重复完成
- 提供稳定收益
- 用于日常资源获取

**隐藏任务**
- 通过探索触发
- 特殊条件解锁
- 奖励稀有物品
- 成就系统关联

### 5.2 任务NPC系统

**基地任务NPC**

1. **指挥官 - 主线任务提供者**
   - 位置：基地指挥中心
   - 任务类型：主线剧情任务
   - 特点：推进核心故事线

2. **情报官 - 情报任务提供者**
   - 位置：基地情报室
   - 任务类型：探索、收集情报
   - 特点：解锁地图信息

3. **训练官 - 训练任务提供者**
   - 位置：基地训练场
   - 任务类型：战斗技巧训练
   - 特点：提升战斗能力

**战斗地图任务NPC**

**刷新机制**
- 每次进入地图随机刷新1-3个任务NPC
- NPC位置在预设刷新点中随机
- 不同NPC有不同刷新权重
- 完成任务后该NPC本次不再刷新

**NPC类型**

1. **幸存者NPC**
   - 刷新率：高（40%）
   - 任务类型：护送、救援、物资收集
   - 奖励：货币、医疗用品
   - 特点：需要保护，可能被敌人攻击

2. **商队NPC**
   - 刷新率：中（25%）
   - 任务类型：护送、清理路线
   - 奖励：货币、交易折扣
   - 特点：完成后可进行交易

3. **研究员NPC**
   - 刷新率：中（20%）
   - 任务类型：收集样本、探索区域
   - 奖励：特殊物品、装备图纸
   - 特点：提供科技相关任务

4. **赏金猎人NPC**
   - 刷新率：低（10%）
   - 任务类型：击杀特定目标、Boss挑战
   - 奖励：高级装备、大量货币
   - 特点：高难度高回报

5. **神秘人NPC**
   - 刷新率：极低（5%）
   - 任务类型：特殊任务、隐藏剧情
   - 奖励：传说装备、独特物品
   - 特点：任务链式，解锁隐藏内容

**NPC交互流程**
1. 发现NPC（地图标记或视觉发现）
2. 靠近NPC触发对话
3. 查看任务描述和奖励
4. 选择接受或拒绝
5. 完成任务目标
6. 返回NPC或自动完成
7. 获得奖励

### 5.3 任务类型详细设计

**击杀类任务**
- 击杀指定数量的敌人
- 击杀特定类型的敌人
- 击杀Boss或精英敌人
- 使用特定武器击杀
- 完成特殊击杀条件（爆头、近战等）

**收集类任务**
- 收集指定物品
- 搜刮特定容器
- 找到隐藏物品
- 收集敌人掉落物
- 收集环境样本

**探索类任务**
- 到达指定地点
- 探索隐藏区域
- 激活特定机关
- 找到线索物品
- 绘制地图区域

**护送类任务**
- 护送NPC到安全点
- 保护NPC不被击杀
- 护送物资到指定地点
- 在限定时间内完成护送

**防御类任务**
- 守住特定区域
- 保护目标物体
- 抵御敌人进攻
- 在限定时间内存活

**解谜类任务**
- 破解密码锁
- 找到钥匙开门
- 按顺序激活机关
- 解读环境线索

### 5.4 任务链系统

**任务链结构**
```
前置任务 → 主任务 → 后续任务1
                  → 后续任务2
                  → 隐藏任务（条件触发）
```

**示例任务链："黑市联络"**

**第一阶段：初次接触**
- 任务：在战斗地图找到黑市联络员
- 目标：探索废弃商场，找到联络员
- 奖励：500货币，黑市情报
- 解锁：黑市商人位置标记

**第二阶段：信任考验**
- 任务：为黑市收集特殊物资
- 目标：收集5个军用物资箱
- 奖励：1000货币，黑市初级权限
- 解锁：黑市商人1级商品

**第三阶段：危险交易**
- 任务：护送黑市货物
- 目标：护送货物到指定地点，击退敌人
- 奖励：2000货币，黑市中级权限
- 解锁：黑市商人2级商品

**第四阶段：核心成员**
- 任务：击杀威胁黑市的敌对势力Boss
- 目标：击杀"掠夺者首领"
- 奖励：5000货币，传说武器，黑市高级权限
- 解锁：黑市商人全部商品，特殊服务

**隐藏分支：黑市秘密**
- 触发条件：在第三阶段发现隐藏线索
- 任务：调查黑市的真实目的
- 奖励：独特装备，隐藏剧情

### 5.5 任务奖励系统

**基础奖励**
- 经验值：根据任务难度
- 货币：根据任务类型
- 商人信誉：提升对应商人关系

**物品奖励**
- 装备：品质和等级根据任务难度
- 消耗品：医疗用品、弹药
- 材料：强化材料、重铸石
- 特殊物品：钥匙、图纸、配方

**解锁奖励**
- 新地图区域
- 新商人
- 新功能
- 新技能

**成就奖励**
- 称号系统
- 特殊外观
- 永久属性加成

## 6. 剧情系统详细设计

### 6.1 剧情表现形式

**对话系统**

**对话界面**
- NPC头像和名称
- 对话文本框
- 选项分支（最多4个选项）
- 语音播放（可选）
- 字幕显示

**对话类型**
- 线性对话：单向信息传递
- 分支对话：玩家选择影响后续
- 条件对话：根据玩家状态变化
- 循环对话：可重复查看的信息

**对话选项影响**
- 任务接受/拒绝
- 信誉变化
- 剧情分支
- 隐藏内容解锁

**CG动画系统**

**CG类型**
- 静态CG：高质量插画
- 动态CG：带有动画效果的插画
- 3D CG：使用游戏引擎渲染的过场
- 混合CG：结合多种表现形式

**CG触发时机**
- 主线任务关键节点
- 重要剧情转折
- Boss战前后
- 结局展示

**CG播放控制**
- 可跳过（已观看过的）
- 可暂停
- 可回看（CG图鉴）
- 自动播放/手动推进

**过场动画系统**

**过场类型**
- 场景转换：基地↔战斗地图
- 剧情过场：推进故事
- 战斗过场：Boss登场
- 结局过场：不同结局展示

**过场动画制作**
- 使用Timeline系统
- 相机动画
- 角色动画
- 特效和音效
- 字幕同步

**动态视频系统**

**视频类型**
- 开场视频：游戏启动时播放
- 剧情视频：重要剧情节点
- 结局视频：不同结局的视频
- 宣传视频：游戏内可查看

**视频播放**
- 全屏播放
- 可跳过（首次不可跳过）
- 字幕支持
- 音量控制

**环境叙事**

**环境细节**
- 场景物品摆放讲述故事
- 文字记录（日记、报告、涂鸦）
- 音频日志
- 环境变化反映剧情进展

**可交互叙事物品**
- 文档：可拾取阅读
- 终端：可查看信息
- 录音：可播放音频
- 照片：可查看图片

### 6.2 剧情结构设计

**主线剧情框架**

**第一章：觉醒**
- 剧情：玩家在灾难后苏醒，加入幸存者基地
- 目标：熟悉基地和基础系统
- 关键事件：
  - 开场CG：灾难发生
  - 教学关卡：基础操作
  - 首次进入战斗地图
  - 首次撤离成功
- 解锁：基础商人、基础装备

**第二章：生存**
- 剧情：探索周边区域，寻找物资和幸存者
- 目标：建立稳定的物资供应链
- 关键事件：
  - 遭遇第一个Boss
  - 解锁新地图区域
  - 发现黑市线索
  - 救援重要NPC
- 解锁：高级商人、新武器类型

**第三章：真相**
- 剧情：调查灾难真相，发现幕后阴谋
- 目标：收集情报，揭开谜团
- 关键事件：
  - 过场动画：真相揭露
  - 重要选择：站队决定
  - 大型战斗：攻坚战
  - 关键NPC死亡/存活
- 解锁：特殊装备、隐藏区域

**第四章：抉择**
- 剧情：面对道德困境，做出关键选择
- 目标：根据选择完成不同任务线
- 关键事件：
  - 分支剧情：不同阵营任务
  - 动态视频：展示选择后果
  - 多个Boss战
  - 基地升级或转移
- 解锁：阵营专属装备

**第五章：终局**
- 剧情：最终决战，迎接结局
- 目标：完成最终任务
- 关键事件：
  - 最终Boss战
  - 结局CG（多结局）
  - 结局视频
  - 后日谈
- 解锁：新游戏+模式

**支线剧情**
- 商人背景故事
- NPC个人任务线
- 隐藏组织调查
- 区域历史探索

### 6.3 剧情分支系统

**选择系统**
- 对话选择影响剧情走向
- 任务完成方式影响结果
- 阵营选择影响可用资源
- 道德选择影响NPC态度

**多结局系统**
- 正义结局：选择保护幸存者
- 黑暗结局：选择个人利益
- 中立结局：保持平衡
- 隐藏结局：满足特殊条件
- 真结局：完成所有隐藏任务

**结局影响因素**
- 主线任务选择
- 关键NPC存活状态
- 阵营信誉
- 特殊任务完成情况
- 收集品完成度

## 7. 战斗系统深度设计

### 7.1 射击机制详细设计

**弹道系统**
- 真实弹道计算
- 子弹飞行时间
- 重力影响
- 风阻影响（可选）
- 穿透系统

**后坐力系统**
- 垂直后坐力
- 水平后坐力
- 后坐力恢复
- 连射后坐力累积
- 配件影响后坐力

**散布系统**
- 基础散布
- 移动散布
- 跳跃散布
- 连射散布增加
- 瞄准减少散布

**瞄准系统**
- 腰射：高散布，快速反应
- 机瞄：中等精度，中等速度
- 瞄具瞄准：高精度，慢速度
- 屏息：短时间完全稳定
- 瞄准晃动：受体力和负重影响
