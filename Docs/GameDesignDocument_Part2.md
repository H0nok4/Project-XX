### 3.4 随机词条系统

**词条分类**

- 精准度：+10% ~ +30% 精准度
- 有效射程：+10% ~ +30% 有效射程

**防御类词条**
- 护甲值加成：+10 ~ +50 护甲值
- 护甲等级：+1 ~ +2 护甲等级
- 耐久加成：+20% ~ +100% 耐久度
- 伤害减免：+3% ~ +15% 伤害减免
- 部位保护：额外保护特定部位

**机动类词条**
- 移动速度：+5% ~ +20% 移动速度

- 切枪速度：+15% ~ +50% 切枪速度
- 瞄准速度：+10% ~ +30% 瞄准速度

- 生命加成：+10% ~ +50% 最大生命
- 体力加成：+10% ~ +50% 最大体力
- 体力恢复：+20% ~ +80% 体力恢复速度
- 治疗效果：+15% ~ +50% 医疗物品效果
- 状态抗性：+10% ~ +40% 异常状态抗性

**特殊类词条**
- 经验加成：+5% ~ +20% 经验获取
- 战利品发现：+10% ~ +30% 稀有物品掉落率
- 弹药回收：击杀有几率回收弹药
- 吸血：造成伤害的1% ~ 5%转化为生命
- 击杀回复：击杀敌人恢复5% ~ 15%生命/体力

**词条品质**
- 词条数值范围根据装备品质决定
- 高品质装备的词条数值更高
- 同一装备不会出现重复词条类型
- 词条可通过重铸系统重新随机

### 3.5 装备技能系统

**技能分类**

**主动技能（需要手动触发）**

1. **战术护盾**
   - 品质要求：史诗+
   - 冷却时间：60秒
   - 效果：展开能量护盾，吸收500点伤害，持续10秒
   - 适用装备：防弹衣

2. **急速换弹**
   - 品质要求：史诗+
   - 冷却时间：30秒

   - 适用装备：武器

3. **战术翻滚**
   - 品质要求：史诗+
   - 冷却时间：15秒

   - 适用装备：战术背心

4. **精准打击**
   - 品质要求：传说
   - 冷却时间：45秒
   - 效果：下一发子弹必定暴击，伤害+200%
   - 适用装备：武器

5. **生命爆发**
   - 品质要求：传说
   - 冷却时间：90秒
   - 效果：瞬间恢复50%生命，并获得10秒伤害免疫30%
   - 适用装备：医疗装备

**被动技能（自动生效）**

1. **击杀回血**
   - 品质要求：稀有+
   - 效果：击杀敌人恢复10%/15%/20%最大生命
   - 适用装备：武器

2. **弹药专家**
   - 品质要求：稀有+
   - 效果：击杀敌人有20%/30%/40%几率回收弹药
   - 适用装备：武器

3. **钢铁之躯**
   - 品质要求：稀有+
   - 效果：受到致命伤害时，有30%/40%/50%几率保留1点生命
   - 适用装备：护甲

4. **战斗狂热**
   - 品质要求：史诗+
   - 效果：连续击杀叠加伤害，每层+5%，最多5层，持续10秒
   - 适用装备：武器

5. **完美闪避**
   - 品质要求：史诗+
   - 效果：有10%几率完全闪避攻击
   - 适用装备：战术背心

6. **嗜血**
   - 品质要求：传说
   - 效果：生命低于30%时，伤害+50%，移速+30%
   - 适用装备：武器

   - 品质要求：传说
   - 效果：生命不会降至1以下，触发后无敌3秒，每次战斗只能触发一次
   - 适用装备：护甲

**技能触发机制**
- 主动技能需要绑定快捷键（如F1-F4）
- 被动技能装备后自动生效
- 同类型技能不叠加，取最高效果
- 技能效果会在HUD上显示

### 3.6 装备强化系统

**强化机制**
- 消耗货币和材料提升装备等级
- 每次强化+1级，最多+5级
- 强化等级影响基础属性
- 强化不影响词条和技能

**强化消耗**
- +1级：1000货币 + 普通材料x5
- +2级：2000货币 + 优秀材料x5
- +3级：4000货币 + 稀有材料x5
- +4级：8000货币 + 史诗材料x5
- +5级：16000货币 + 传说材料x5

**强化效果**
- 每级提升10%基础属性
- +5级装备基础属性+50%

### 3.7 装备改造系统

**词条重铸**
- 消耗重铸石重新随机词条
- 保持词条数量不变
- 词条品质范围不变
- 可能获得更好或更差的词条

**词条锁定**
- 消耗锁定石锁定特定词条
- 锁定的词条在重铸时不变
- 最多锁定2个词条
- 锁定消耗随词条数量增加

**品质提升**
- 消耗大量材料提升装备品质
- 普通→优秀：5000货币 + 材料
- 优秀→稀有：15000货币 + 材料
- 稀有→史诗：50000货币 + 材料
- 史诗→传说：不可提升（传说只能掉落）

### 4.2 战斗地图商人系统

**隐藏商人机制**

**解锁条件**
- 完成特定任务链
- 找到隐藏区域
- 使用特殊钥匙
- 达到特定信誉等级

**商人类型**

1. **黑市商人**
   - 位置：地图隐蔽角落
   - 解锁：完成"黑市联络"任务链
   - 特色：出售违禁品、高风险高回报物品

   - 位置：随机刷新
   - 解锁：随机遭遇
   - 特色：限时出现，商品随机
   - 商品：传说装备、独特物品、稀有材料

   - 位置：废弃工厂
   - 解锁：完成"寻找工匠"任务
   - 特色：提供装备改造服务

   - 位置：安全屋

   - 特色：出售地图信息、任务线索

**战斗地图交易风险**

- 可能被敌人打断
- 商人可能被击杀（导致无法交易）
- 价格比基地商人贵20-50%
- 但能获得基地买不到的物品
