# Project-XX 游戏总体设计文档

## 1. 游戏概述

### 1.1 游戏定位
Project-XX 是一款单机第一人称搜打撤射击游戏，融合了《逃离塔科夫》的核心玩法和深度RPG系统。玩家需要在危险的战斗地图中探索、战斗、完成任务，并将战利品带回基地进行角色成长。

### 1.2 核心玩法循环
```
基地准备 → 配置装备 → 进入战斗地图 → 探索/战斗/任务 → 撤离/死亡 → 结算 → 基地成长
```

### 1.3 游戏特色
- **双场景结构**：基地安全区 + 战斗地图危险区
- **风险收益机制**：死亡丢失装备，撤离保留战利品
- **深度RPG系统**：装备等级、品质、随机词条、技能系统
- **多层次商人**：基地固定商人 + 战斗地图隐藏商人
- **剧情驱动**：任务链、CG、过场动画、动态视频
- **动态任务NPC**：战斗地图中随机刷新的任务提供者

## 2. 场景系统

### 2.1 基地场景（局外/Meta层）

#### 场景定位
基地是玩家的安全区域，是所有局外活动的中心。玩家从基地出发进入战斗地图，撤离后返回基地。

#### 核心功能区域

**仓库区**
- 安全仓库（Warehouse Stash）：永久存储，不受死亡影响
- 武器柜（Weapon Locker）：存放未装备的武器
- 装备展示区：查看已拥有的装备

**商人区**
- 多个固定位置的商人NPC
- 靠近商人后按E键打开商店界面
- 每个商人有独立的商品目录和信誉系统

**准备区**
- 装备配置界面：配置进入战斗地图的装备
- 战斗背包（Raid Backpack）：风险区，死亡清空
- 装备槽位：主武器、副武器、近战武器、护甲
- 安全箱（Secure Container）：保护区，死亡保留
- 特殊装备槽（Special Equipment）：保护区，用于任务道具和钥匙

**任务区**
- 任务公告板：查看可接取的任务
- NPC对话：与任务NPC交互
- 任务追踪：查看进行中的任务

**成长区**
- 角色属性界面
- 技能树系统
- 装备强化工作台
- 装备改造工作台

#### 基地商人系统

**商人类型**
1. **武器商人（Weapon Dealer）**
   - 商品：各类武器、弹药、配件
   - 位置：基地武器区
   - 初始等级：1级

2. **护甲商人（Armor Dealer）**
   - 商品：头盔、防弹衣、战术背心
   - 位置：基地防护区
   - 初始等级：1级

3. **医疗商人（Medical Dealer）**
   - 商品：医疗用品、药品、注射器
   - 位置：基地医疗区
   - 初始等级：1级

4. **杂货商人（General Dealer）**
   - 商品：消耗品、任务道具、钥匙
   - 位置：基地中心区
   - 初始等级：1级

**商人成长系统**
- 商人等级：通过交易额提升（每10000货币提升1级）
- 信誉系统：完成商人委托任务提升信誉
- 等级解锁：
  - 1级：基础商品
  - 2级：优秀品质商品
  - 3级：稀有品质商品
  - 4级：史诗品质商品
  - 5级：传说品质商品
- 信誉折扣：
  - 中立：无折扣
  - 友好：95折
  - 尊敬：90折
  - 崇拜：85折

**商人交互流程**
1. 玩家靠近商人（距离<2米）
2. 显示交互提示："按E与[商人名称]交易"
3. 按E键打开商店界面
4. 显示商人等级、信誉、商品列表
5. 购买/出售物品
6. 关闭界面返回基地

### 2.2 战斗地图（局内/Raid层）

#### 场景定位
战斗地图是高风险高回报的核心玩法区域。玩家需要在限定时间内完成目标并撤离。

#### 核心玩法要素

**探索与搜刮**
- 地面战利品：随机刷新的物品
- 容器搜刮：箱子、柜子、保险箱
- 隐藏区域：需要钥匙或完成任务解锁
- 环境交互：门、可破坏物、机关

**战斗系统**
- AI敌人：多种类型和难度
- Boss战：特殊强力敌人
- 尸体搜刮：击杀敌人后搜刮装备
- 战术要素：掩体、噪声、视野

**任务系统**
- 任务NPC：随机刷新在地图中
- 任务链：完成前置任务解锁后续任务
- 任务目标：击杀、收集、探索、护送
- 任务奖励：经验、货币、装备、解锁商人

**商人系统**
- 隐藏商人：需要完成任务链解锁
- 特殊商品：稀有装备和独特物品
- 风险交易：在危险区域进行交易
- 限时刷新：商品有刷新周期

**撤离系统**
- 撤离点：地图中的安全撤离区域
- 撤离条件：到达撤离点并等待倒计时
- 多个撤离点：提供战术选择
- 撤离状态：成功撤离保留所有风险区物品

#### 风险机制

**风险区（死亡丢失）**
- 战斗背包（Raid Backpack）：主要搜刮容器
- 主武器（Primary Weapon）：可在局内更换
- 副武器（Secondary Weapon）：可在局内更换
- 护甲（Armor）：头盔、防弹衣等

**保护区（死亡保留）**
- 近战武器（Melee Weapon）：永不丢失
- 安全箱（Secure Container）：小容量保护容器
- 特殊装备（Special Equipment）：任务道具、钥匙

**死亡惩罚**
- 清空所有风险区物品
- 保留保护区物品
- 损失部分经验值（可选）
- 任务进度保留

**撤离奖励**
- 保留风险区当前所有物品
- 保留保护区物品
- 获得任务奖励
- 获得经验值

#### 时间限制
- 每个战斗地图有时间限制（如30-45分钟）
- 超时未撤离视为失败
- 时间剩余会影响敌人刷新强度

## 3. RPG装备系统

### 3.1 装备分类

**武器类**
- 主武器：步枪、冲锋枪、霰弹枪、狙击枪
- 副武器：手枪、微型冲锋枪
- 近战武器：刀、斧、棍棒

**护甲类**
- 头盔：保护头部
- 防弹衣：保护躯干
- 战术背心：额外携带容量

**配件类**
- 瞄具：红点、全息、倍镜
- 枪口：消音器、补偿器、制退器
- 握把：前握把、战术握把
- 弹匣：扩容弹匣、快速弹匣

**消耗品类**
- 医疗：绷带、医疗包、注射器
- 弹药：各类口径子弹
- 投掷物：手雷、闪光弹、烟雾弹

### 3.2 装备品质系统

**品质等级**
1. **普通（Common）** - 白色
   - 基础属性
   - 无额外词条
   - 掉落率：60%

2. **优秀（Uncommon）** - 绿色
   - 基础属性 +10%
   - 1个随机词条
   - 掉落率：25%

3. **稀有（Rare）** - 蓝色
   - 基础属性 +25%
   - 2个随机词条
   - 掉落率：10%

4. **史诗（Epic）** - 紫色
   - 基础属性 +50%
   - 3个随机词条
   - 可能带有1个技能
   - 掉落率：4%

5. **传说（Legendary）** - 橙色
   - 基础属性 +100%
   - 4个随机词条
   - 必定带有1-2个技能
   - 掉落率：1%

### 3.3 装备等级系统

**等级范围**
- 装备等级：1-50级
- 玩家等级：1-50级
- 等级要求：装备有最低等级要求

**等级影响**
- 基础属性随等级提升
- 高等级装备有更高的词条数值
- 等级差距影响战斗效果

**等级获取**
- 战斗地图掉落：根据地图难度决定等级范围
- 商人购买：商人等级影响出售装备等级
- 任务奖励：任务难度决定奖励等级
- 装备强化：可提升装备等级（最多+5级）
